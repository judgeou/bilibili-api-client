# Bilibili 视频下载工具

## 使用

需要 [node](https://nodejs.org/) >= 16

推荐使用 [pnpm](https://pnpm.io/zh/) 代替 npm

```sh
npm install
```

```sh
npm run start
```

控制台会出现二维码，使用哔哩哔哩手机APP扫码登陆。

随后输入BV号，即可下载视频。（已支持完整网址、多P下载、新番下载）
